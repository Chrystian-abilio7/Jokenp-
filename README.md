 <h1>🎮 <b>Jokenpô (Rock Paper Scissors)</b></h1>

A browser-based Rock Paper Scissors game built using HTML, CSS and JavaScript.
The player competes against the computer, with a score system and a fun pixel-style interface.

🚀 <b>Features</b>

- ✊ Rock, ✋ Paper, ✌️ Scissors choices
- 🤖 Random computer choice
- 📊 Score tracking for player and computer
- 🎨 Pixel-style background interface
- 🔄 Reset button to restart the game
- 🧠 Game logic implemented in JavaScript
🛠️ Technologies Used

HTML5

CSS3

JavaScript (Vanilla JS)

🎯 How to Play

Choose between Rock, Paper, or Scissors.

The computer will randomly choose its move.

The winner is decided based on classic rules:

Rock beats Scissors

Scissors beats Paper

Paper beats Rock

The score updates after each round.
